### Directory Structure

The sources of the project follows this structure:

```
/src
  /redux
    /{domain}
      /{domain}.actions.tsx
      /{domain}.reducer.tsx
      /{domain}.selector.tsx
      /{domain}.types.tsx
      /{domain}.utils.tsx
  /components
    /{component}
      /{component}.tsx
      /{component}.styles.scss
      /index.tsx
      /types.tsx
 /pages
    /{page}
      {page}.tsx
      {page}.styles.tsx
      types.tsx
      index.tsx
      
```
